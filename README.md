# ⚡ QuickQR

A modern **QR Code Generator Web App** built using **HTML, CSS, and JavaScript**.  
Generate customized QR codes instantly with multiple content types and download them in seconds.

---

## 🚀 Live Demo

👉 Use the app online:  
https://shubhamkale01-dev.github.io/QuickQR/

*(If the link is not working yet, enable GitHub Pages in repo settings.)*

---

## ✨ Features

- Generate QR codes instantly
- Custom QR colors
- Adjustable QR size
- Add logo inside QR
- Download QR code (PNG / SVG)
- Modern UI with animated tabs

### Supported QR Types
- 🌐 Website
- 📄 PDF
- 📱 App Downloads
- 👤 Contact Card
- 🔗 Link Page
- 📝 Text
- ✉️ Email
- 💬 SMS

## 📥 Download & Run Locally

1. Click **Code**
2. Click **Download ZIP**
3. Extract the folder
4. Open **index.html**

The app will run directly in your browser.
